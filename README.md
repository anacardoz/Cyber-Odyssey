# 🏛️ Prólogo

Bem-vindo ao **Cyber Odyssey**.

Depois de alguns meses estudando Segurança Cibernética, percebi que conhecia muitos conceitos, mas ainda precisava fortalecer minha base.

Foi então que decidi reorganizar meus estudos, aprofundar os fundamentos e documentar toda essa jornada.

Este repositório é esse registro.

Aqui você encontrará minhas anotações, laboratórios, desafios, aprendizados e projetos desenvolvidos ao longo do caminho.

Meu objetivo é mostrar minha evolução.

---

# ⚓ Por que "Cyber Odyssey"?

A ideia nasceu da **Odisseia**, uma das maiores histórias da mitologia grega.

Após a Guerra de Troia, Odisseu inicia uma longa viagem até Ítaca.

Durante essa jornada ele enfrenta desafios, aprende com seus erros e passa por diferentes ilhas antes de chegar ao seu destino.

Vejo minha carreira da mesma forma.

Cada novo conhecimento representa um passo adiante.

Cada projeto concluído representa uma nova experiência.

# 📖 Os Livros

Minha jornada será organizada em livros.

Cada livro representa uma etapa importante da construção da minha base em Segurança Cibernética.

Atualmente meus estudos estão focados em:

- 🌐 Redes
- 🐧 Linux
- 🪟 Windows
- 🛡️ Segurança da Informação
- 📜 Logs
- 📡 Wireshark
- 👁️ SIEM
- 📊 Wazuh / Splunk
- 🧩 MITRE ATT&CK
- 🧪 Laboratórios
- 🐙 Git & GitHub

Novos capítulos serão adicionados conforme minha jornada continuar.

# 📂 Projetos

Os projetos desenvolvidos durante esta jornada ficarão organizados dentro deste repositório.

# 📓 Diário de Bordo

Além dos projetos, também registro minhas experiências durante os estudos.

Cada entrada do diário inclui:

- O que aprendi

# 🏛️ Ítaca

Na Odisseia, Ítaca representa o destino.

Na minha jornada, Ítaca representa a construção de uma carreira sólida em Segurança Cibernética.

Ainda não sei exatamente quais dificuldades encontrarei pelo caminho.

Mas sei que cada capítulo escrito neste repositório me deixará um pouco mais próximo desse objetivo.
